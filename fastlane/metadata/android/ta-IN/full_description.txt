<i>PCAPdroid</i> என்பது தனியுரிமைக்கு ஏற்றப் பயன்பாடாகும், இது உங்கள் சாதனத்தில் உள்ள பிற பயன்பாடுகளால் செய்யப்பட்ட இணைப்புகளைக் கண்காணிக்கவும் பகுப்பாய்வு செய்யவும் உங்களை அனுமதிக்கிறது.

இது போக்குவரத்தின் PCAP டம்பை ஏற்றுமதி செய்யவும், HTTP ஐ ஆய்வு செய்யவும், TLS போக்குவரத்தை டிக்ரிப்ட் செய்யவும் மற்றும் பலவற்றையும் செய்ய உங்களை அனுமதிக்கிறது.

ரூட் இல்லாமல் நெட்வொர்க் டிராஃபிக்கைப் பிடிக்க PCAPdroid ஒரு VPN ஐ உருவகப்படுத்துகிறது. இது தொலைநிலை VPN சேவையகத்தைப் பயன்படுத்தாது, அதற்குப் பதிலாகத் தரவு சாதனத்தில் உள்ளூரில் செயலாக்கப்படும்.

<b>அம்சங்கள்:</b>

* பயனர்கள் மற்றும் சிஸ்டம் பயன்பாடுகளால் செய்யப்பட்ட இணைப்புகளைப் பதிவுசெய்து ஆய்வு செய்யுங்கள்
* SNI, DNS வினவல், HTTP URL மற்றும் தொலை IP முகவரியைப் பிரித்தெடுக்கவும்.
* உள்ளமைக்கப்பட்ட டிகோடர்களுக்கு நன்றி HTTP கோரிக்கைகள் மற்றும் பதில்களை ஆய்வு செய்யுங்கள்
* முழு இணைப்பு பேலோடையும் ஹெக்ஸ்டம்ப்/டெக்ஸ்ட் ஆக ஆய்வு செய்யவும்
* HTTPS/TLS போக்குவரத்தை டிக்ரிப்ட் செய்து SSLKEYLOGFILE ஐ ஏற்றுமதி செய்யவும்
* நிகழ்நேர பகுப்பாய்விற்காக (எ.கா. வயர்ஷார்க்) டிராஃபிக்கை PCAP கோப்பிற்கு மாற்றவும், உலாவியிலிருந்து பதிவிறக்கவும் அல்லது ரிமோட் ரிசீவருக்கு ஸ்ட்ரீம் செய்யவும்.
* நல்ல போக்குவரத்தை வடிகட்டவும், முரண்பாடுகளை எளிதாகக் கண்டறியவும் விதிகளை உருவாக்குங்கள்.
* ஆஃப்லைன் DB தேடல்கள்மூலம் ரிமோட் சர்வரின் நாடு மற்றும் ASN ஐ அடையாளம் காணவும்
* ரூட் செய்யப்பட்ட சாதனங்களில், பிற VPN பயன்பாடுகள் இயங்கும்போது போக்குவரத்தைப் பிடிக்கவும்
